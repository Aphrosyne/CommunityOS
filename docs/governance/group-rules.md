# Group Rules 群规

> **状态：** 正式
> **版本：** v1.0
> **最后更新：** 2026-06-30

# 目的

本文档为社区群组规则，是所有管理员操作与自动化处理的基础执行依据。

群规原文（可供直接复制使用）见文末。

---

# 核心禁令

1. 严禁群成员以任何形式对其他成员实施"开盒"行为（包括但不限于人肉搜索、泄露个人真实姓名、电话、住址、身份证号、工作单位等隐私信息），此类行为已涉嫌违反《网络安全法》《个人信息保护法》等法律法规，坚决零容忍。
2. 严禁借群聊平台组织、教唆、协助他人实施上述违法违规行为，或传播、扩散他人隐私信息。

---

# 责任声明

1. 本群管理员及群主仅负责维护群内基础交流秩序，已明确告知上述禁令并将尽力制止违规行为，但无法实时监控所有成员的私下行为及自发操作。
2. 若任何群成员违反本禁令，实施"开盒"等违法违规行为，均属于涉事成员个人自发行为，其法律责任、民事赔偿责任等全部由涉事方独立承担，与本群管理员、群主及群本身无任何关联。
3. 管理员及群主从未授权、教唆任何成员实施此类行为，也不对涉事成员的个人行为承担任何连带责任，受害方应直接向涉事个人追责或向相关执法部门报案。

---

# 违规处理

1. 一经发现或核实存在"开盒"相关行为，立即移出本群，永久拉黑，并不予任何解释。
2. 若因涉事成员行为引发法律纠纷，本群管理方将积极配合执法部门调查，提供群内可查的相关记录，但不承担任何额外法律责任。

---

# 附则

1. 本条例自发布之日起生效，所有群成员加入即视为认可并遵守。
2. 群管理方有权根据实际情况补充或修订条例，修订后将在群内公示。

---

# 交流规范

1. 本群仅用于 mod 与整合包相关交流，严禁讨论、传播其他 MOD / 整合包作者的付费作品。
2. 严禁发布散兵相关内容、色情（含 sex、诱导性词汇、低俗文本）、暴力、R18G、涉政及易引发争吵的节奏性内容，违者立即移出群聊。
3. 安装及使用整合包前，请完整通读下载链接内教程与所有文档，严格按指引完成安装、更新操作，不要在群里询问安装教程以及必看文档有的内容，大概率没人会理你。
4. 整合包相关问题优先查阅文档及公告，闪退等异常情况 95% 为操作不当导致，剩余多与设备配置相关，优先自查。
5. 群资源有限，每月将固定清理不活跃成员。
6. 在提问的时候，做好冷场的心理准备，没人有义务必须回答你的问题，提问时请准确描述问题，最好携带问题截图。
7. 本群规最终解释权归管理员及群主所有，违反群规者不予辩驳，直接处理。
